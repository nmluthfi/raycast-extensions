# Nansen Changelog

## [Initial Release] - {PR_MERGE_DATE}

- Initial release of the Nansen Raycast Extension.
- Includes predictive market intelligence search via **Search Markets** tool.
- Introduces robust insider finding capabilities with the **Scan Insiders** command for Polymarket.
- Incorporates wallet analysis functionality, allowing users to map out connected entities using **Cluster Wallet**.
- Provides features to get wallet labels and on-chain PnL data directly inline via **Get Wallet Positions** and **Get Wallet Labels**.
