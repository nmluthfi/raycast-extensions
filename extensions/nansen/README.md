# Nansen

Prediction market intelligence powered by Nansen's onchain analytics. Discover insiders, reveal wallet clusters, and investigate cross-market behavior on Polymarket.

## Getting Started

To use this extension, you will need a Nansen API key. 
You can get one at [Nansen Agent Setup](https://app.nansen.ai/auth/agent-setup).

## Features

- **Search Markets**: Search for Polymarket prediction markets.
- **Scan Insiders**: Scan markets for wallets with suspicious trading patterns.
- **Cluster Wallet**: Find related wallets for a given address.
- **Get Wallet Positions**: Get active and recent trading positions.
- **Get Wallet Labels**: Get Nansen labels and categories.
